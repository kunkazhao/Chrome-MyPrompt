# 赵二的提示词 (Chrome-MyPrompt)

一个优雅、高效的 Chrome 侧边栏提示词（Prompts）管理插件。基于最新 **Manifest V3** 规范构建，无需依赖任何第三方重型框架，零编译、秒级启动、极致轻量。

本插件采用了高阶 **米色暖雅风毛玻璃拟态 (Warm Beige Glassmorphism)** 视觉设计，不仅在视觉上温润素雅，还集成了非常高频实用的交互设计。

---

## 🌟 核心功能特性

* **🌾 暖米色美学设计**：乳白透亮毛玻璃卡片、细腻的金铜色渐变交互、精致的淡金投影与 `10px` 方圆圆角，全面适配现代高质感 UI 追求。
* **📌 全局快捷置顶**：
  - 默认展示最左侧的「置顶」分类，将高频使用的提示词一键收藏在此。
  - 在置顶列表中，支持二次确认取消置顶，防止误触。
* **🖐️ 双向拖拽排序 (Drag & Drop)**：
  - **提示词排序**：在普通分类下，你可以直接通过鼠标/手势拖动提示词卡片来任意调整位置。
  - **分类重排**：在「管理分类」弹窗中，支持使用手柄拖拽自定义分类的展示顺序，首页 Tab 栏会实时刷新保持同步。
* **⚡ 一键极速复制**：卡片上配备专属复制按钮，一键将提示词存入系统剪贴板，并滑出金铜色精美 Toast 提示。
* **🗂️ 分类管理与安全删除**：
  - 支持分类的新增、重命名和删除。
  - 当您删除某个物理分类时，原分类下的所有提示词将**自动归入当前剩余的第一个物理分类中**，提供绝佳的数据防空防丢保护。
* **💻 鼠标拖拽横向滚动 (Drag-to-scroll)**：针对没有触控板的 PC 鼠标党，在首页分类 Tab 栏按住鼠标左右拉拽即可丝滑横移滚动分类，极度顺手。
* **📚 预置 29 个专业级生产力提示词**：
  - **通用/置顶**：包含旅游文案、图片优化、参数图/封面图设计等 5 个高频提示词。
  - **文案与小红书**：小红书标题/正文爆款仿写、千字文大师、智能金句、文本润色纠错专家。
  - **公众号运营**：内置 `公众号写作专家`、`微信公众号文章HTML生成器`。
  - **程序开发**：`开发-Goal模式`、`代码重构专家`、`SQL 语句生成与优化器`、`Git Commit Message 生成器`、`代码 Bug 定位与修复专家`。
  - **视频与音乐**：`Suno AI 歌词创作大师`、`Sora 与 Veo 视频生成元提示词`。

## 🔧 安装与使用指引

### 1. 克隆或下载本仓库
```bash
git clone https://github.com/kunkazhao/Chrome-MyPrompt.git
```
或者直接下载 ZIP 压缩包并解压到本地。

### 2. 在 Chrome 中加载插件
1. 打开 **Chrome 浏览器**，在地址栏输入并回车：`chrome://extensions/`（或者点击右上角三点 -> 扩展程序 -> 管理扩展程序）。
2. 在扩展页面右上角，开启 **「开发者模式」** 开关。
3. 点击页面左上角的 **「加载已解压的扩展程序」**。
4. 选择刚刚下载/克隆的项目文件夹（即包含 `manifest.json` 的目录）。

### 3. 打开侧边栏
1. 加载成功后，建议在 Chrome 浏览器右上角点击 🧩 拼图图标，将 **「赵二的提示词」** 📌 固定到工具栏。
2. 点击工具栏上的插件图标（红底“词”字图标），右侧将自动滑出充满毛玻璃科技感的米色侧边栏。

